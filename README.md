Cordova (PhoneGap) 3.0+ Plugin to connect to Google's native Universal Analytics SDK

# Installing

This plugin follows the Cordova 3.0+ plugin spec, so it can be installed through the Cordova CLI in your existing Cordova project:
```bash
cordova plugin add https://github.com/jbailey223/google-analytics-plugin.git
```

# Note

This modification of the google analytics cordova plugin was created for use with other plugins that require version 9.8.0 of the google play services and should not be used instead of the original plugin under normal circumstances.

See original documentation for instructions:
https://github.com/danwilson/google-analytics-plugin
